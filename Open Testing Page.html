Panduan Open Testing
iSmartOffice Patient

Panduan singkat dan praktis untuk tester bergabung, menginstal, dan memberikan feedback pada aplikasi iSmartOffice Patient melalui Google Play Open Testing.

Apa itu Open Testing?

Open Testing adalah pengujian publik di Google Play. Siapa pun yang memiliki tautan opt-in dapat bergabung sebagai tester dan menginstal aplikasi versi uji langsung dari Play Store.

Langkah Inti Open Testing

1. Siapkan Build

Build AAB rilis, versionCode dinaikkan, target API terbaru.

2. Setup Track

Play Console → Testing → Open Testing → Create Release.

3. Upload & Rollout

Upload AAB, isi release notes, dan mulai rollout.

4. Bagikan Link

Sebarkan tautan opt-in ke calon tester.

5. Pantau & Evaluasi

Kumpulkan feedback, crash report, dan ANR.

Cara Tester Install Aplikasi

  1. Scan QR Code Open Testing di samping menggunakan kamera HP.
  2. Tap tombol “Jadi penguji” (login dengan akun Google yang sama).
  3. Setelah berhasil opt-in, aplikasi akan terbuka di Google Play Store.
  4. Tap Install atau Update dan aktifkan pembaruan otomatis.

Catatan: Untuk keluar dari program uji, buka kembali link Open Testing lalu pilih “Keluar dari program uji”, kemudian uninstall aplikasi.
